# Mario Multiverse Fan-made Launcher
An fanmade Launcher built on Godot 4.6, for a fangame made by neoarcutus Mario Multiverse, currently W.I.P

Currently only Booting up the game, editor, game settings, updating the launcher is supported check the upcoming roadmap for more details

## OS Support
Only Windows and Linux operating systems are currently supported MacOS support might be added when i can
